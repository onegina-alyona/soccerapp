## **SoccerApp (MERN: Mongoose, Express, ReactJS, NodeJs)**

DB: MongoDB

Backend: Express(NodeJS)

Frontend: ReactJS

- **npm install**
- Execute locally
  - Run NodeJS Server

_cd backend_

_npm start_

Backend server will be running on port 4000

  - Run Frontend

_cd frontend_

_npm start_

- Build App
  - Build the app

_npm run build_

- Database Name

**soccerDB**
